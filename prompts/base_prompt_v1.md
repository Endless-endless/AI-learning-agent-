# Base Prompt

本文件用于记录我在 Day 3 学到的 Prompt 基本结构与可复用示例。

核心原则：
Prompt 不是提问，而是对 AI 的行为设计。

---

## Prompt 基本结构

一个完整的 Prompt，至少应包含：

1. 角色（Role）
2. 目标（Goal）
3. 输入信息（Input）
4. 约束条件（Constraints）
5. 输出形式（Output Format）

下面是我的实践示例。

## 示例 1：概念解释型 Prompt

你是一名擅长给初学者讲解 AI 概念的老师。

目标：
用通俗、准确的方式解释一个 AI 概念。

输入：
概念名称：激活函数
我的当前理解水平：基础直觉

约束：
- 不使用公式
- 不使用学术化语言
- 使用类比帮助理解

输出：
- 1 段直觉解释
- 1 个生活化类比

## 示例 2：结构化学习 Prompt

你是一名 AI 学习教练。

目标：
帮助我系统理解一个概念。

输入：
概念：大语言模型

约束：
- 按固定结构输出
- 每一部分不超过 3 行

输出格式：
1. 一句话定义
2. 工作机制
3. 常见误解
4. 使用时应注意的问题

## 示例 3：纠错与改进 Prompt

你是一名严格的 AI 教练。

目标：
检查并改进我的理解。

输入：
这是我对“激活值”的理解：
「激活值是神经元的输出结果，由加权求和和激活函数计算得到。」

约束：
- 指出不严谨或模糊的地方
- 不直接否定，而是给出改进建议

输出：
1. 问题点
2. 改进建议
3. 优化后的版本
