
# Contributing Guide

欢迎对本项目提出建议或改进。

---

## 贡献方式

- 提交 Issue：问题 / 讨论 / 改进建议
- 提交 Pull Request：文档优化 / 结构改进

---

## 文档规范

### Markdown
- 清晰标题层级
- 代码块标注语言
- 合理段落间距

### 命名规范
- Day 文件：DayX_topic_name.md
- Prompt 文件：Prompt_xxx.md

---

## 内容原则

- 说明设计原因
- 避免只给结论
- 鼓励结构化表达

---

## Commit 建议

Add DayX content  
Refine prompt structure  
Improve agent evaluation logic
