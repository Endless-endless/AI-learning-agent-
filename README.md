# AI Learning Agent 🚀

> 一个以 **长期学习 + 工程化记录** 为核心的 AI 学习项目

这是我系统学习人工智能（重点是 **大语言模型 / Agent / AI 应用**）的个人项目仓库。

这个项目 **不是速成教程，也不是工具集合**，  
而是我用工程化方式记录 **AI 学习路径、思考过程与认知变化** 的地方。

---

## 🙋‍♂️ 我的学习立场

- 我不把 AI 当作“魔法”或“权威”
- 我更关心 **AI 是如何工作的，而不是它看起来有多聪明**
- 我选择用 **自己的语言** 理解 AI，而不是复制他人的结论

这个仓库记录的是：  
> **一个普通学习者，如何一步步建立对 AI 的真实理解**

---

## 🎯 项目目标

- 建立对 AI（尤其是大语言模型）的**正确、可解释的理解**
- 用 **自己的语言** 而不是复制资料来学习 AI
- 把零散学习过程沉淀为 **可回溯、可复用的认知资产**
- 为后续构建 **AI Agent / AI 应用** 打下扎实基础

---

## 🚫 项目边界

为了保证学习质量，这个项目 **刻意不做**：

- ❌ 只堆 API 示例、不理解原理
- ❌ 不加思考地复制博客 / 课程内容
- ❌ 追逐“最新最火”而忽略基础
- ❌ 一上来就做复杂系统或产品

我更看重：  
> **慢一点，但每一步都知道“为什么”。**
